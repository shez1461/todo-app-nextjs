## Features
**Task Module**
- Create Task
- Edit Task
- Delete Task
- Mark as Completed

## Bonus
* Integration tests
* React Hooks
* Persist tasks to disk
* Redux

## Technologies
- [React](https://reactjs.org/)
- [NextJs](https://nextjs.org) 
- [MaterialUI](https://mui.com/)

## Run App
```bash
npm i
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
